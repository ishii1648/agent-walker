# Agent Instructions

このリポジトリでは、Codex は原則として日本語で応答する。

## 言語

- ユーザーへの説明、進捗報告、最終回答は日本語で書く。
- `docs/` 配下の仕様書、設計書、TODO、開発メモは日本語で書く。
- README も基本は日本語で書く。
- コード、識別子、ファイル名、コミットメッセージは英語でよい。
- 外部 API 名、ライブラリ名、UI ラベル、一般的な技術用語は自然な場合のみ英語のままでよい。

## ドキュメント

- 仕様、設計、TODO は日本語を基本とする。
- 必要に応じて英語の用語を併記してよい。
- プロダクト判断や設計理由も、後から読み返せるように日本語で残す。

## コミュニケーション

- 変更内容は簡潔な日本語で説明する。
- 実装上の判断や未解決事項がある場合は、短く明示する。
